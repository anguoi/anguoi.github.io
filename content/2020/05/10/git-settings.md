## 查看全局配置

```
git config --global --list
```

## 设置全局用户名

```
git config --global user.name "vhxubo"
```

## 设置全局邮箱

```
git config --global user.email "vhxubo@gmail.com"
```

## 创建 SSH Key

```
ssh-keygen -t rsa -C "vhxubo@gmail.com"
```
